# 错误处理与泛型

> **目标**：掌握 Go 的错误处理惯用法（error 接口、包装、哨兵错误），理解 panic/recover 的使用场景，以及 Go 1.18+ 泛型。

---

## 第一部分：错误处理

### 一、error 接口

```go
type error interface {
    Error() string
}
```

Go 的错误就是一个实现了 `Error()` 方法的值，没有异常机制。

### 1.1 创建错误

```go
// 方式 1：errors.New
err := errors.New("something went wrong")

// 方式 2：fmt.Errorf（支持格式化）
err := fmt.Errorf("user %d not found", userID)

// 方式 3：自定义错误类型
type NotFoundError struct {
    Resource string
    ID       int
}
func (e *NotFoundError) Error() string {
    return fmt.Sprintf("%s #%d not found", e.Resource, e.ID)
}
```

### 1.2 错误处理范式

```go
result, err := doSomething()
if err != nil {
    return fmt.Errorf("doSomething failed: %w", err) // 包装错误
}
// 使用 result
```

**黄金法则**：检查每一个返回的 error，要么处理，要么向上传播（包装后 return）。

### 二、错误包装与解包

#### 2.1 包装（%w）

```go
func readConfig(path string) ([]byte, error) {
    data, err := os.ReadFile(path)
    if err != nil {
        return nil, fmt.Errorf("readConfig(%s): %w", path, err)
    }
    return data, nil
}
// 错误信息: readConfig(/etc/app.conf): open /etc/app.conf: no such file or directory
```

#### 2.2 检查 errors.Is

```go
err := readConfig("/etc/app.conf")
if errors.Is(err, os.ErrNotExist) {
    // 文件不存在，创建默认配置
}
// errors.Is 会递归解包 %w 链，查找目标错误
```

#### 2.3 提取 errors.As

```go
var pathErr *os.PathError
if errors.As(err, &pathErr) {
    fmt.Println("failed path:", pathErr.Path)
}
// errors.As 会递归解包，找到匹配类型的错误
```

### 三、哨兵错误 vs 自定义类型

```go
// 哨兵错误：预定义的错误值（简单场景）
var ErrNotFound = errors.New("not found")
var ErrConflict = errors.New("conflict")

// 自定义类型：需要携带额外信息时
type ValidationError struct {
    Field   string
    Message string
}
func (e *ValidationError) Error() string {
    return fmt.Sprintf("validation: %s - %s", e.Field, e.Message)
}
```

### 四、panic 和 recover

```go
// panic: 程序遇到不可恢复的错误
func mustParseURL(raw string) *url.URL {
    u, err := url.Parse(raw)
    if err != nil {
        panic(fmt.Sprintf("invalid URL: %s", raw))
    }
    return u
}

// recover: 只能在 defer 中使用
func safeHandler(w http.ResponseWriter, r *http.Request) {
    defer func() {
        if r := recover(); r != nil {
            log.Printf("panic recovered: %v\n%s", r, debug.Stack())
            http.Error(w, "Internal Server Error", 500)
        }
    }()
    handleRequest(w, r) // 可能 panic
}
```

**原则**：
- 只在**真正不可恢复**的情况下 panic（配置解析、必须成功的初始化）
- 不要用 panic 做流程控制
- 库代码**不应该** panic，应该返回 error

### 五、错误处理最佳实践

```go
// 1. 只处理一次
// ❌ 日志记录后又返回
if err != nil {
    log.Println(err)
    return err  // 上层可能再记一次
}
// ✅ 只做一件事
if err != nil {
    return fmt.Errorf("fetch user: %w", err)
}

// 2. 提供上下文
// ❌ 无上下文
return err
// ✅ 有上下文
return fmt.Errorf("save order #%d: %w", orderID, err)

// 3. 在顶层统一处理
func main() {
    if err := run(); err != nil {
        log.Fatal(err)
    }
}
```

## 第二部分：泛型（Go 1.18+）

### 六、类型参数基础

```go
// 泛型函数
func Min[T int | float64 | string](a, b T) T {
    if a < b {
        return a
    }
    return b
}

Min(3, 5)        // T=int，推断
Min(3.14, 2.71)  // T=float64
```

### 6.1 类型约束

```go
// 内联约束
func Add[T int | float64](a, b T) T { return a + b }

// 定义约束接口
type Number interface {
    int | int8 | int16 | int32 | int64 |
    float32 | float64
}
func Sum[T Number](nums []T) T {
    var total T
    for _, n := range nums {
        total += n
    }
    return total
}
```

### 6.2 内置约束

```go
import "golang.org/x/exp/constraints" // 或使用 cmp 包

// comparable: 支持 == 和 !=
func Contains[T comparable](slice []T, target T) bool {
    for _, v := range slice {
        if v == target { return true }
    }
    return false
}

// any: 任意类型（= interface{}）
func Print[T any](v T) { fmt.Println(v) }

// cmp.Ordered (Go 1.21): 支持 < > <= >=
import "cmp"
func Max[T cmp.Ordered](a, b T) T {
    if a > b { return a }
    return b
}
```

### 七、泛型数据结构

#### 7.1 泛型栈

```go
type Stack[T any] struct {
    data []T
}

func (s *Stack[T]) Push(v T) {
    s.data = append(s.data, v)
}

func (s *Stack[T]) Pop() (T, bool) {
    if len(s.data) == 0 {
        var zero T
        return zero, false
    }
    v := s.data[len(s.data)-1]
    s.data = s.data[:len(s.data)-1]
    return v, true
}

func (s *Stack[T]) Peek() (T, bool) {
    if len(s.data) == 0 {
        var zero T
        return zero, false
    }
    return s.data[len(s.data)-1], true
}

// 使用
s := &Stack[int]{}
s.Push(1); s.Push(2)
v, _ := s.Pop() // 2
```

#### 7.2 泛型链表

```go
type Node[T any] struct {
    Value T
    Next  *Node[T]
}

type LinkedList[T any] struct {
    Head *Node[T]
    Size int
}

func (l *LinkedList[T]) PushFront(v T) {
    l.Head = &Node[T]{Value: v, Next: l.Head}
    l.Size++
}
```

### 八、泛型工具函数

```go
// Map: 对切片每个元素执行转换
func Map[T, U any](s []T, f func(T) U) []U {
    result := make([]U, len(s))
    for i, v := range s {
        result[i] = f(v)
    }
    return result
}

// Filter: 过滤切片
func Filter[T any](s []T, pred func(T) bool) []T {
    var result []T
    for _, v := range s {
        if pred(v) {
            result = append(result, v)
        }
    }
    return result
}

// Reduce: 归约
func Reduce[T, U any](s []T, init U, f func(U, T) U) U {
    acc := init
    for _, v := range s {
        acc = f(acc, v)
    }
    return acc
}

// 使用
nums := []int{1, 2, 3, 4, 5}
doubled := Map(nums, func(n int) int { return n * 2 })       // [2,4,6,8,10]
evens := Filter(nums, func(n int) bool { return n%2 == 0 })  // [2,4]
sum := Reduce(nums, 0, func(acc, n int) int { return acc+n })// 15
```

### 九、slices 和 maps 包（Go 1.21+）

```go
import "slices"

// 排序
slices.Sort(nums)
slices.SortFunc(users, func(a, b User) int {
    return cmp.Compare(a.Age, b.Age)
})

// 查找
idx, found := slices.BinarySearch(nums, 42)
ok := slices.Contains(nums, 42)

// 其他
slices.Reverse(nums)
slices.Compact(nums) // 去除连续重复

import "maps"
maps.Keys(m)    // 返回所有 key
maps.Values(m)  // 返回所有 value
maps.Clone(m)   // 浅拷贝
```

## 速查表

| 概念 | 语法/要点 |
|------|----------|
| 创建错误 | `errors.New()` / `fmt.Errorf()` |
| 包装错误 | `fmt.Errorf("ctx: %w", err)` |
| 判断错误 | `errors.Is(err, target)` |
| 提取错误 | `errors.As(err, &target)` |
| panic | 仅用于不可恢复的场景 |
| recover | 只在 defer 中生效 |
| 泛型函数 | `func F[T constraint](x T) T` |
| 约束 | `any` / `comparable` / `cmp.Ordered` / 自定义 |
| 泛型类型 | `type Stack[T any] struct{...}` |
| 零值 | `var zero T`（泛型中获取零值） |
