## 一、Go 语言设计哲学

Go 由 Google 三位大神（Ken Thompson、Rob Pike、Robert Griesemer）在 2007 年设计，2012 年发布 1.0。

**核心理念**：

| 原则 | 体现 |
|------|------|
| 简单 | 关键字只有 25 个（C 有 32，Java 有 50+） |
| 高效 | 编译型语言，原生支持并发 |
| 实用 | 内置 goroutine/channel，标准库覆盖面广 |
| 统一 | `gofmt` 强制统一代码风格，无需争论 |

## 二、程序结构

### 2.1 Hello World

```go
package main          // 每个文件必须声明所属包；main 包是入口

import "fmt"          // 导入标准库

func main() {         // main 函数是程序入口
    fmt.Println("Hello, World!")
}
```

### 2.2 包（Package）

```go
// 一个文件夹 = 一个包
// 包名通常与文件夹名相同
package mypackage

// 大写开头 = 导出（public）
func PublicFunc() {}

// 小写开头 = 包内私有（private）
func privateFunc() {}
```

**导入规则**：
```go
import (
    "fmt"                     // 标准库
    "math/rand"               // 标准库子包
    "github.com/gin-gonic/gin" // 第三方包
    myalias "long/package/path" // 别名
    _ "driver/mysql"          // 只执行 init()，不直接使用
)
```

## 三、类型系统

### 3.1 基本类型

```go
// 布尔
var b bool = true

// 整数
var i int = 42      // 平台相关（64位系统上为int64）
var i8 int8         // -128 ~ 127
var i16 int16       // -32768 ~ 32767
var i32 int32       // -2^31 ~ 2^31-1
var i64 int64       // -2^63 ~ 2^63-1
var u uint          // 无符号
var u8 uint8        // 0 ~ 255（= byte）
var u32 uint32      // 0 ~ 2^32-1（= rune 的底层类型）

// 浮点
var f32 float32     // ~7 位有效数字
var f64 float64     // ~15 位有效数字（推荐默认使用）

// 复数
var c64 complex64
var c128 complex128

// 字符串
var s string = "hello"

// byte 和 rune
var ch byte = 'A'     // uint8，表示 ASCII
var r rune = '中'     // int32，表示 Unicode 码点
```

### 3.2 零值（Zero Value）

Go 的每个类型都有明确的零值，不存在"未初始化"的变量。

| 类型 | 零值 |
|------|------|
| bool | `false` |
| 数值类型 | `0` |
| string | `""` |
| 指针/slice/map/chan/func/interface | `nil` |

### 3.3 类型转换

Go **不做隐式类型转换**，必须显式：

```go
var i int = 42
var f float64 = float64(i)   // int → float64
var u uint = uint(f)          // float64 → uint

// 字符串 ↔ 字节切片
s := "hello"
bs := []byte(s)     // string → []byte
s2 := string(bs)    // []byte → string

// 字符串 ↔ rune 切片
rs := []rune("你好")  // string → []rune
s3 := string(rs)      // []rune → string

// 整数 ↔ 字符串
import "strconv"
str := strconv.Itoa(42)         // int → string  "42"
num, _ := strconv.Atoi("42")    // string → int  42
```

## 四、变量声明

### 4.1 四种方式

```go
// 方式1: var + 类型（完整声明）
var name string = "Go"

// 方式2: var + 类型推断（省略类型）
var name = "Go"

// 方式3: 短声明（只能在函数内）
name := "Go"

// 方式4: 批量声明
var (
    name   string = "Go"
    version int   = 22
    stable  bool  = true
)
```

### 4.2 常量

```go
const Pi = 3.14159
const (
    StatusOK    = 200
    StatusNotFound = 404
)

// iota：自动递增的常量生成器
const (
    Sunday    = iota  // 0
    Monday            // 1
    Tuesday           // 2
    Wednesday         // 3
)

// iota 高级用法
const (
    _  = iota             // 0，丢弃
    KB = 1 << (10 * iota) // 1 << 10 = 1024
    MB                    // 1 << 20
    GB                    // 1 << 30
    TB                    // 1 << 40
)
```

## 五、控制流

### 5.1 if

```go
// 标准形式
if x > 0 {
    fmt.Println("positive")
} else if x == 0 {
    fmt.Println("zero")
} else {
    fmt.Println("negative")
}

// 带初始化语句（变量作用域仅限 if 块）
if err := doSomething(); err != nil {
    fmt.Println("error:", err)
}
// err 在这里不可访问
```

### 5.2 for

Go 只有 `for`，没有 `while` 和 `do-while`：

```go
// 经典三段式
for i := 0; i < 10; i++ {
    fmt.Println(i)
}

// while 风格
for condition {
    // ...
}

// 无限循环
for {
    // ...
    if done { break }
}

// range 遍历
for index, value := range slice {
    fmt.Println(index, value)
}

for key, value := range myMap {
    fmt.Println(key, value)
}

for index, char := range "Hello" {
    fmt.Printf("%d: %c\n", index, char)
}

// 只要下标
for i := range slice { ... }

// 只要值
for _, v := range slice { ... }
```

### 5.3 switch

```go
// 基本 switch（不需要 break，默认不穿透）
switch day {
case "Monday":
    fmt.Println("周一")
case "Tuesday", "Wednesday":
    fmt.Println("周二或周三")
default:
    fmt.Println("其他")
}

// 无条件 switch（替代 if-else 链）
switch {
case score >= 90:
    grade = "A"
case score >= 80:
    grade = "B"
default:
    grade = "C"
}

// type switch（接口类型断言）
switch v := val.(type) {
case int:
    fmt.Println("int:", v)
case string:
    fmt.Println("string:", v)
}

// fallthrough（显式穿透到下一个 case）
switch n {
case 1:
    fmt.Println("one")
    fallthrough
case 2:
    fmt.Println("two or after one")
}
```

### 5.4 defer

`defer` 语句延迟到函数返回前执行。多个 defer 按 **LIFO（后进先出）** 顺序。

```go
func readFile(path string) error {
    f, err := os.Open(path)
    if err != nil {
        return err
    }
    defer f.Close() // 无论函数怎么退出，都会关闭文件

    // 读取文件...
    return nil
}

// 多个 defer 的执行顺序
func demo() {
    defer fmt.Println("1")
    defer fmt.Println("2")
    defer fmt.Println("3")
}
// 输出: 3, 2, 1
```

**典型用途**：关闭文件、释放锁、关闭连接、记录日志。

## 六、函数

### 6.1 基本语法

```go
// 基本函数
func add(a, b int) int {
    return a + b
}

// 多返回值
func divide(a, b float64) (float64, error) {
    if b == 0 {
        return 0, fmt.Errorf("division by zero")
    }
    return a / b, nil
}

// 命名返回值
func swap(a, b int) (x, y int) {
    x = b
    y = a
    return // 裸 return，自动返回 x, y
}

// 可变参数
func sum(nums ...int) int {
    total := 0
    for _, n := range nums {
        total += n
    }
    return total
}
// 调用: sum(1, 2, 3) 或 sum(slice...)
```

### 6.2 函数是一等公民

```go
// 函数赋值给变量
add := func(a, b int) int { return a + b }
fmt.Println(add(1, 2))

// 函数作为参数
func apply(f func(int, int) int, a, b int) int {
    return f(a, b)
}

// 闭包
func counter() func() int {
    n := 0
    return func() int {
        n++
        return n
    }
}
c := counter()
fmt.Println(c()) // 1
fmt.Println(c()) // 2
fmt.Println(c()) // 3
```

### 6.3 init 函数

每个包可以有多个 `init()` 函数，在 `main()` 之前自动执行：

```go
func init() {
    // 初始化配置、注册驱动等
}
```

**执行顺序**：被依赖的包的 init → 当前包的 init → main()

## 七、运算符

```go
// 算术: + - * / %
// 比较: == != < > <= >=
// 逻辑: && || !
// 位运算: & | ^ << >> &^

// 特有: &^ (AND NOT，位清除)
x := 0b1010
y := 0b1100
fmt.Printf("%04b\n", x&^y) // 0010 (清除 x 中 y 为 1 的位)
```

## 速查表

| 概念 | 要点 |
|------|------|
| 零值 | 所有变量自动初始化 |
| 大小写 | 大写=导出，小写=私有 |
| 类型转换 | 必须显式 `T(v)` |
| for | Go 唯一的循环语句 |
| switch | 默认不穿透，需要 `fallthrough` |
| defer | LIFO 顺序，函数退出前执行 |
| 多返回值 | `func f() (int, error)` |
| 闭包 | 函数捕获外部变量 |
| iota | 常量自动递增器 |
