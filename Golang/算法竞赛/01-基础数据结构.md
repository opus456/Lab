## 一、栈（Stack）

### 1.1 核心概念

栈是**后进先出**（LIFO）的数据结构。想象一摞盘子：你只能从最上面放盘子、取盘子。

```
        ┌───┐
push →  │ 3 │  ← top (最后放的，最先取)
        ├───┤
        │ 2 │
        ├───┤
        │ 1 │  ← bottom (最先放的，最后取)
        └───┘
```

**三个操作**：
- `Push(x)`：把 x 放到顶部
- `Pop()`：取出并返回顶部元素
- `Top()/Peek()`：看一眼顶部元素但不取出

### 1.2 Go 实现

Go 没有内置栈，用切片实现极其简洁：

```go
// === 栈模板 ===
type Stack []int

func (s *Stack) Push(v int) { *s = append(*s, v) }
func (s *Stack) Pop() int   { v := (*s)[len(*s)-1]; *s = (*s)[:len(*s)-1]; return v }
func (s *Stack) Top() int   { return (*s)[len(*s)-1] }
func (s *Stack) Empty() bool { return len(*s) == 0 }

// 使用
var stk Stack
stk.Push(1)
stk.Push(2)
fmt.Println(stk.Top()) // 2
stk.Pop()               // 移除 2
```

**竞赛中更常见的简写**（不封装，直接用切片）：

```go
stk := []int{}
stk = append(stk, 1)            // push
stk = append(stk, 2)            // push
top := stk[len(stk)-1]          // peek
stk = stk[:len(stk)-1]          // pop
```

### 1.3 经典应用：括号匹配

**题目**：给定一个只含 `()[]{}` 的字符串，判断括号是否匹配。

```go
func isValid(s string) bool {
    stk := []byte{}
    pairs := map[byte]byte{')': '(', ']': '[', '}': '{'}

    for i := 0; i < len(s); i++ {
        c := s[i]
        if c == '(' || c == '[' || c == '{' {
            stk = append(stk, c) // 左括号入栈
        } else {
            // 右括号：检查栈顶是否匹配
            if len(stk) == 0 || stk[len(stk)-1] != pairs[c] {
                return false
            }
            stk = stk[:len(stk)-1] // 匹配成功，弹出
        }
    }
    return len(stk) == 0 // 栈空 = 全部匹配
}
```

### 1.4 经典应用：单调栈

**问题**：对数组中每个元素，找到**下一个比它大的元素**。

```
输入:  [2, 1, 2, 4, 3]
输出:  [4, 2, 4, -1, -1]
解释:  2 的下一个更大 → 4; 1 的下一个更大 → 2; ...
```

**核心思路**：维护一个**从栈底到栈顶递减**的栈（存下标）。遍历时，如果当前元素比栈顶大，说明栈顶找到了答案。

```go
func nextGreater(nums []int) []int {
    n := len(nums)
    ans := make([]int, n)
    for i := range ans {
        ans[i] = -1
    }

    stk := []int{} // 存下标

    for i := 0; i < n; i++ {
        // 当前元素比栈顶对应的值大 → 栈顶元素找到答案
        for len(stk) > 0 && nums[i] > nums[stk[len(stk)-1]] {
            top := stk[len(stk)-1]
            stk = stk[:len(stk)-1]
            ans[top] = nums[i]
        }
        stk = append(stk, i)
    }
    return ans
}
```

**直觉**：栈里存的是"还没找到答案的人"。一旦来了个更大的数，所有比它小的"等待者"都找到了答案，出栈。

**时间复杂度**：O(N)。每个元素最多入栈、出栈各一次。

## 二、队列（Queue）

### 2.1 核心概念

队列是**先进先出**（FIFO）的数据结构。想象排队买票：先来的人先买。

```
入队 → [1] [2] [3] → 出队
        ↑           ↑
      back        front
```

### 2.2 Go 实现

```go
// 方式1: 切片（简单但不释放已出队的内存）
queue := []int{}
queue = append(queue, 1)    // 入队 (enqueue)
queue = append(queue, 2)
front := queue[0]           // 查看队头
queue = queue[1:]           // 出队 (dequeue)

// 方式2: 使用 container/list（双向链表，真正 O(1) 出队）
import "container/list"
q := list.New()
q.PushBack(1)               // 入队
q.PushBack(2)
front := q.Front().Value    // 查看队头
q.Remove(q.Front())         // 出队
```

**竞赛中推荐方式1**，因为代码最短。内存问题在竞赛中几乎不影响。

### 2.3 经典应用：BFS（广度优先搜索）

**题目**：在一个 N×M 的网格中，从 (0,0) 到 (N-1,M-1) 的最短路径长度。0 表示可通行，1 表示障碍。

```go
func shortestPath(grid [][]int) int {
    n, m := len(grid), len(grid[0])
    if grid[0][0] == 1 || grid[n-1][m-1] == 1 {
        return -1
    }

    type Point struct{ x, y int }
    dx := [4]int{-1, 1, 0, 0}
    dy := [4]int{0, 0, -1, 1}

    dist := make([][]int, n)
    for i := range dist {
        dist[i] = make([]int, m)
        for j := range dist[i] {
            dist[i][j] = -1
        }
    }

    queue := []Point{{0, 0}}
    dist[0][0] = 0

    for len(queue) > 0 {
        cur := queue[0]
        queue = queue[1:]

        for d := 0; d < 4; d++ {
            nx, ny := cur.x+dx[d], cur.y+dy[d]
            if nx >= 0 && nx < n && ny >= 0 && ny < m &&
                grid[nx][ny] == 0 && dist[nx][ny] == -1 {
                dist[nx][ny] = dist[cur.x][cur.y] + 1
                if nx == n-1 && ny == m-1 {
                    return dist[nx][ny]
                }
                queue = append(queue, Point{nx, ny})
            }
        }
    }
    return dist[n-1][m-1]
}
```

### 2.4 双端队列（Deque）

两头都能进出的队列。Go 中一般用切片模拟：

```go
type Deque []int

func (d *Deque) PushFront(v int) { *d = append([]int{v}, (*d)...) }
func (d *Deque) PushBack(v int)  { *d = append(*d, v) }
func (d *Deque) PopFront() int   { v := (*d)[0]; *d = (*d)[1:]; return v }
func (d *Deque) PopBack() int    { v := (*d)[len(*d)-1]; *d = (*d)[:len(*d)-1]; return v }
func (d *Deque) Front() int      { return (*d)[0] }
func (d *Deque) Back() int       { return (*d)[len(*d)-1] }
func (d *Deque) Len() int        { return len(*d) }
```

**经典应用：滑动窗口最大值**

```go
// 给定数组和窗口大小 k，输出每个窗口的最大值
func maxSlidingWindow(nums []int, k int) []int {
    dq := []int{}   // 存下标，对应值从大到小
    result := []int{}

    for i := 0; i < len(nums); i++ {
        // 移除超出窗口的元素
        for len(dq) > 0 && dq[0] <= i-k {
            dq = dq[1:]
        }
        // 维护单调递减：比当前值小的都没用了
        for len(dq) > 0 && nums[dq[len(dq)-1]] <= nums[i] {
            dq = dq[:len(dq)-1]
        }
        dq = append(dq, i)

        if i >= k-1 {
            result = append(result, nums[dq[0]])
        }
    }
    return result
}
```

## 三、链表（Linked List）

### 3.1 核心概念

数组在内存中连续存储，链表的每个节点散布在内存各处，通过"指针"串联。

```
数组: [1][2][3][4]  连续内存，随机访问 O(1)，插入删除 O(N)

链表: [1]→[2]→[3]→[4]→nil  分散内存，访问 O(N)，插入删除 O(1)
```

| 操作 | 数组 | 链表 |
|------|------|------|
| 按下标访问 | O(1) | O(N) |
| 在头部插入 | O(N) | O(1) |
| 在任意位置插入 | O(N) | O(1)（已知位置） |
| 删除 | O(N) | O(1)（已知位置） |

### 3.2 Go 实现

```go
type ListNode struct {
    Val  int
    Next *ListNode
}

// 创建链表 1→2→3
head := &ListNode{Val: 1}
head.Next = &ListNode{Val: 2}
head.Next.Next = &ListNode{Val: 3}

// 遍历
for p := head; p != nil; p = p.Next {
    fmt.Println(p.Val)
}
```

### 3.3 经典操作：反转链表

```go
func reverseList(head *ListNode) *ListNode {
    var prev *ListNode
    curr := head
    for curr != nil {
        next := curr.Next  // 先保存下一个
        curr.Next = prev   // 反转指针
        prev = curr        // prev 前进
        curr = next        // curr 前进
    }
    return prev
}
```

**图解**：
```
初始:  1 → 2 → 3 → nil
       ↑
      curr, prev=nil

Step1: nil ← 1    2 → 3 → nil
             ↑    ↑
            prev  curr

Step2: nil ← 1 ← 2    3 → nil
                  ↑    ↑
                 prev  curr

Step3: nil ← 1 ← 2 ← 3
                       ↑
                      prev (新的 head)
```

### 3.4 经典操作：判断链表有环（快慢指针）

```go
func hasCycle(head *ListNode) bool {
    slow, fast := head, head
    for fast != nil && fast.Next != nil {
        slow = slow.Next       // 慢指针走1步
        fast = fast.Next.Next  // 快指针走2步
        if slow == fast {
            return true // 相遇 = 有环
        }
    }
    return false // fast 到了末尾 = 无环
}
```

**直觉**：快指针追慢指针，就像操场上跑步。如果有环（跑道），快的人一定会追上（套圈）。如果没环（直线），快的人先到终点。

### 3.5 虚拟头节点技巧

很多链表题需要特殊处理 head 边界。引入一个 dummy 节点可以统一处理：

```go
func removeElements(head *ListNode, val int) *ListNode {
    dummy := &ListNode{Next: head}
    prev := dummy
    for curr := head; curr != nil; curr = curr.Next {
        if curr.Val == val {
            prev.Next = curr.Next // 跳过当前节点
        } else {
            prev = curr
        }
    }
    return dummy.Next // dummy.Next 就是真正的 head
}
```

## 四、堆 / 优先队列（Heap / Priority Queue）

### 4.1 核心概念

堆是一棵**完全二叉树**，满足堆性质：
- **最小堆**：每个节点 ≤ 它的子节点 → 根节点最小
- **最大堆**：每个节点 ≥ 它的子节点 → 根节点最大

```
最小堆:          最大堆:
      1                9
     / \              / \
    3   2            5   8
   / \              / \
  5   4            1   3
```

**用数组存储**（利用完全二叉树的性质）：

```
下标:     0  1  2  3  4
          1  3  2  5  4

父节点:   (i-1)/2
左子节点: 2*i+1
右子节点: 2*i+2
```

| 操作 | 时间复杂度 |
|------|----------|
| 取最值 | O(1) |
| 插入 | O(log N) |
| 删除最值 | O(log N) |

### 4.2 Go 的 container/heap

Go 标准库提供了 `container/heap` 接口。你需要实现 5 个方法：

```go
import "container/heap"

// 最小堆
type MinHeap []int

func (h MinHeap) Len() int           { return len(h) }
func (h MinHeap) Less(i, j int) bool { return h[i] < h[j] } // < 最小堆, > 最大堆
func (h MinHeap) Swap(i, j int)      { h[i], h[j] = h[j], h[i] }
func (h *MinHeap) Push(x any)        { *h = append(*h, x.(int)) }
func (h *MinHeap) Pop() any {
    old := *h
    v := old[len(old)-1]
    *h = old[:len(old)-1]
    return v
}

// 使用
h := &MinHeap{3, 1, 4, 1, 5}
heap.Init(h)               // 建堆 O(N)
heap.Push(h, 2)            // 插入
min := heap.Pop(h).(int)   // 取出最小值: 1
```

### 4.3 最大堆（只需改 Less）

```go
type MaxHeap []int

func (h MaxHeap) Len() int           { return len(h) }
func (h MaxHeap) Less(i, j int) bool { return h[i] > h[j] } // 注意这里是 >
func (h MaxHeap) Swap(i, j int)      { h[i], h[j] = h[j], h[i] }
func (h *MaxHeap) Push(x any)        { *h = append(*h, x.(int)) }
func (h *MaxHeap) Pop() any {
    old := *h
    v := old[len(old)-1]

```

### 4.4 自定义结构体堆

```go
type Edge struct {
    to, weight int

type EdgeHeap []Edge

func (h EdgeHeap) Len() int           { return len(h) }

    return v

```

### 4.5 经典应用：Top K 问题

**题目**：从 N 个数中找第 K 大的数。

**方法1：最小堆，维护大小为 K**

```go
func findKthLargest(nums []int, k int) int {
    h := &MinHeap{}
    heap.Init(h)

    for _, v := range nums {
        heap.Push(h, v)
        if h.Len() > k {
            heap.Pop(h) // 堆大小超过K，弹出最小的
        }
    }
    // 堆中剩 K 个最大的数，堆顶就是第 K 大
    return heap.Pop(h).(int)
}
```

**时间复杂度**：O(N log K)

### 4.6 经典应用：合并 K 个有序链表

```go
type NodeHeap []*ListNode
func (h NodeHeap) Len() int           { return len(h) }
func (h NodeHeap) Less(i, j int) bool { return h[i].Val < h[j].Val }
func (h NodeHeap) Swap(i, j int)      { h[i], h[j] = h[j], h[i] }
func (h *NodeHeap) Push(x any)        { *h = append(*h, x.(*ListNode)) }
func (h *NodeHeap) Pop() any {
    old := *h; v := old[len(old)-1]; *h = old[:len(old)-1]; return v
}

func mergeKLists(lists []*ListNode) *ListNode {
    h := &NodeHeap{}
    for _, l := range lists {
        if l != nil {
            heap.Push(h, l)
        }
    }

    dummy := &ListNode{}
    tail := dummy

    for h.Len() > 0 {
        node := heap.Pop(h).(*ListNode)
        tail.Next = node
        tail = node
        if node.Next != nil {
            heap.Push(h, node.Next)
        }
    }
    return dummy.Next
}
```

## 五、哈希表（Hash Map）

### 5.1 核心概念

哈希表通过**哈希函数**将键(key)映射到数组下标，实现近乎 O(1) 的查找、插入、删除。

```
key "apple"  → hash("apple") → 下标 3 → 存入 array[3]
key "banana" → hash("banana") → 下标 7 → 存入 array[7]
```

**冲突处理**：不同的 key 可能映射到同一个下标（哈希冲突），常见方案：
- **链地址法**：每个下标挂一个链表
- **开放寻址法**：冲突时探测下一个空位

Go 的 `map` 内部使用链地址法。

### 5.2 竞赛中的常见用法

```go
// 1. 频率统计
freq := map[int]int{}
for _, v := range nums {
    freq[v]++
}

// 2. 去重
seen := map[int]bool{}
for _, v := range nums {
    if !seen[v] {
        seen[v] = true
        // 第一次见到 v
    }
}

// 3. 两数之和
func twoSum(nums []int, target int) []int {
    mp := map[int]int{} // 值 → 下标
    for i, v := range nums {
        if j, ok := mp[target-v]; ok {
            return []int{j, i}
        }
        mp[v] = i
    }
    return nil
}

// 4. 用 map 实现集合操作
setA := map[int]bool{1: true, 2: true, 3: true}
setB := map[int]bool{2: true, 3: true, 4: true}
// 交集
for k := range setA {
    if setB[k] { fmt.Println(k) }
}
```

## 六、总结对比

| 数据结构 | 适用场景 | Go 实现方式 |
|---------|---------|------------|
| 栈 | 括号匹配、单调栈、DFS | `[]int` 切片 |
| 队列 | BFS、层序遍历 | `[]int` 切片 |
| 双端队列 | 滑动窗口 | `[]int` 切片 |
| 链表 | 频繁插入删除 | `struct` + 指针 |
| 堆 | Top K、Dijkstra、合并 | `container/heap` |
| 哈希表 | 查找、计数、去重 | `map[K]V` |
