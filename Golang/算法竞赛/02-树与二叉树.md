# 树与二叉树

> **目标**：彻底理解树结构的递归本质，掌握所有遍历方式和经典算法，建立"看到树就想到递归"的直觉。

---

## 一、树的基本概念

### 1.1 什么是树？

树是一种**层次结构**。生活中的例子：
- 文件系统（文件夹套文件夹）
- 公司组织架构（CEO → VP → 经理 → 员工）
- HTML DOM（html → body → div → p）

```
         A          ← 根节点 (root)
       / | \
      B  C  D       ← A 的子节点
     / \    |
    E   F   G       ← 叶子节点 (leaf): 没有子节点
```

| 术语 | 含义 |
|------|------|
| 根节点 | 没有父节点的节点（树的起点） |
| 叶子节点 | 没有子节点的节点 |
| 深度 | 从根到该节点的边数 |
| 高度 | 从该节点到最远叶子的边数 |
| 子树 | 某个节点及其所有后代 |

### 1.2 二叉树

每个节点**最多两个子节点**（左子、右子）的树。

```go
type TreeNode struct {
    Val   int
    Left  *TreeNode
    Right *TreeNode
}
```

**特殊二叉树**：

| 类型 | 定义 | 图示 |
|------|------|------|
| 满二叉树 | 每层都是满的 | 每个非叶节点都有两个子 |
| 完全二叉树 | 除最后一层外全满，最后一层从左到右 | 堆就是完全二叉树 |
| 二叉搜索树(BST) | 左 < 根 < 右（对所有节点递归成立） | 中序遍历是升序 |
| 平衡二叉树 | 任意节点左右子树高度差 ≤ 1 | AVL 树 |

## 二、二叉树遍历（最核心的技能）

### 2.1 四种遍历

```
        1
       / \
      2   3
     / \
    4   5
```

| 遍历方式 | 顺序 | 结果 | 记忆口诀 |
|---------|------|------|---------|
| 前序 | **根**→左→右 | 1,2,4,5,3 | 根在**前** |
| 中序 | 左→**根**→右 | 4,2,5,1,3 | 根在**中** |
| 后序 | 左→右→**根** | 4,5,2,3,1 | 根在**后** |
| 层序 | 按层从左到右 | 1,2,3,4,5 | BFS |

**"前中后"指的是根节点的访问时机。**

### 2.2 递归实现（最直观）

```go
// 前序遍历
func preorder(root *TreeNode, result *[]int) {
    if root == nil {
        return
    }
    *result = append(*result, root.Val)  // 先访问根
    preorder(root.Left, result)          // 再左
    preorder(root.Right, result)         // 再右
}

// 中序遍历
func inorder(root *TreeNode, result *[]int) {
    if root == nil {
        return
    }
    inorder(root.Left, result)           // 先左
    *result = append(*result, root.Val)  // 再根
    inorder(root.Right, result)          // 再右
}

// 后序遍历
func postorder(root *TreeNode, result *[]int) {
    if root == nil {
        return
    }
    postorder(root.Left, result)         // 先左
    postorder(root.Right, result)        // 再右
    *result = append(*result, root.Val)  // 最后根
}
```

**递归的本质**：每次函数调用就是"进入一个子树"，`return` 就是"退出子树回到父节点"。

### 2.3 非递归实现（用栈模拟）

```go
// 前序遍历（非递归）
func preorderIterative(root *TreeNode) []int {
    if root == nil {
        return nil
    }
    result := []int{}
    stack := []*TreeNode{root}

    for len(stack) > 0 {
        node := stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = append(result, node.Val)

        // 先压右再压左（因为栈是后进先出，左先出）
        if node.Right != nil {
            stack = append(stack, node.Right)
        }
        if node.Left != nil {
            stack = append(stack, node.Left)
        }
    }
    return result
}

// 中序遍历（非递归）
func inorderIterative(root *TreeNode) []int {
    result := []int{}
    stack := []*TreeNode{}
    curr := root

    for curr != nil || len(stack) > 0 {
        // 一路向左走到底
        for curr != nil {
            stack = append(stack, curr)
            curr = curr.Left
        }
        // 弹出（访问节点）
        curr = stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        result = append(result, curr.Val)
        // 转向右子树
        curr = curr.Right
    }
    return result
}
```

### 2.4 层序遍历（BFS）

```go
func levelOrder(root *TreeNode) [][]int {
    if root == nil {
        return nil
    }
    result := [][]int{}
    queue := []*TreeNode{root}

    for len(queue) > 0 {
        size := len(queue) // 当前层的节点数
        level := []int{}

        for i := 0; i < size; i++ {
            node := queue[0]
            queue = queue[1:]
            level = append(level, node.Val)

            if node.Left != nil {
                queue = append(queue, node.Left)
            }
            if node.Right != nil {
                queue = append(queue, node.Right)
            }
        }
        result = append(result, level)
    }
    return result
}
```

## 三、递归思维：解决树问题的通用框架

### 3.1 核心框架

绝大多数树问题的递归解法遵循一个模式：

```go
func solve(root *TreeNode) ReturnType {
    // 1. Base Case（递归终止条件）
    if root == nil {
        return 某个初始值
    }

    // 2. 递归处理子问题
    leftResult := solve(root.Left)
    rightResult := solve(root.Right)

    // 3. 合并结果
    return combine(root.Val, leftResult, rightResult)
}
```

### 3.2 例题：求二叉树的最大深度

```go
func maxDepth(root *TreeNode) int {
    if root == nil {
        return 0 // 空树深度为 0
    }
    left := maxDepth(root.Left)   // 左子树深度
    right := maxDepth(root.Right) // 右子树深度
    return max(left, right) + 1   // 取较大的 + 1（加上自己这层）
}
```

**递归展开过程**：
```
maxDepth(1)
├─ maxDepth(2)
│  ├─ maxDepth(4) → left=0, right=0 → return 1
│  └─ maxDepth(5) → left=0, right=0 → return 1
│  → max(1,1)+1 = 2
└─ maxDepth(3) → left=0, right=0 → return 1
→ max(2,1)+1 = 3
```

### 3.3 例题：判断对称二叉树

```go
func isSymmetric(root *TreeNode) bool {
    return isMirror(root.Left, root.Right)
}

func isMirror(a, b *TreeNode) bool {
    if a == nil && b == nil { return true }
    if a == nil || b == nil { return false }
    return a.Val == b.Val &&
        isMirror(a.Left, b.Right) &&  // 外侧对比
        isMirror(a.Right, b.Left)     // 内侧对比
}
```

### 3.4 例题：二叉树的直径

**直径**：任意两个节点之间的最长路径的边数。不一定过根节点。

```go
func diameterOfBinaryTree(root *TreeNode) int {
    ans := 0

    var depth func(*TreeNode) int
    depth = func(node *TreeNode) int {
        if node == nil {
            return 0
        }
        l := depth(node.Left)
        r := depth(node.Right)
        // 经过当前节点的最长路径 = 左深度 + 右深度
        if l+r > ans {
            ans = l + r
        }
        return max(l, r) + 1
    }

    depth(root)
    return ans
}
```

**关键洞察**：每个节点作为"拐点"时，经过它的路径长度 = 左深度 + 右深度。遍历所有节点取最大值。

## 四、二叉搜索树（BST）

### 4.1 定义

对于 BST 中的每个节点：
- 左子树所有值 < 当前值
- 右子树所有值 > 当前值

```
        8
       / \
      3   10
     / \    \
    1   6    14
       / \   /
      4   7 13
```

**核心性质**：BST 的中序遍历结果是升序的。

### 4.2 查找

```go
func searchBST(root *TreeNode, target int) *TreeNode {
    if root == nil || root.Val == target {
        return root
    }
    if target < root.Val {
        return searchBST(root.Left, target)  // 去左边找
    }
    return searchBST(root.Right, target)      // 去右边找
}
```

**时间复杂度**：平衡时 O(log N)，退化为链表时 O(N)。

### 4.3 插入

```go
func insertBST(root *TreeNode, val int) *TreeNode {
    if root == nil {
        return &TreeNode{Val: val}
    }
    if val < root.Val {
        root.Left = insertBST(root.Left, val)
    } else {
        root.Right = insertBST(root.Right, val)
    }
    return root
}
```

### 4.4 删除

删除是 BST 操作中最复杂的，分三种情况：

```go
func deleteBST(root *TreeNode, key int) *TreeNode {
    if root == nil {
        return nil
    }
    if key < root.Val {
        root.Left = deleteBST(root.Left, key)
    } else if key > root.Val {
        root.Right = deleteBST(root.Right, key)
    } else {
        // 找到了要删除的节点
        if root.Left == nil {
            return root.Right  // case1: 无左子 → 右子顶替
        }
        if root.Right == nil {
            return root.Left   // case2: 无右子 → 左子顶替
        }
        // case3: 有两个子 → 用右子树最小值替换
        successor := root.Right
        for successor.Left != nil {
            successor = successor.Left
        }
        root.Val = successor.Val
        root.Right = deleteBST(root.Right, successor.Val)
    }
    return root
}
```

### 4.5 BST 相关经典问题

**验证 BST**：

```go
func isValidBST(root *TreeNode) bool {
    return validate(root, math.MinInt64, math.MaxInt64)
}

func validate(node *TreeNode, minVal, maxVal int) bool {
    if node == nil {
        return true
    }
    if node.Val <= minVal || node.Val >= maxVal {
        return false
    }
    return validate(node.Left, minVal, node.Val) &&
        validate(node.Right, node.Val, maxVal)
}
```

**BST 第 K 小元素**（中序遍历第 K 个）：

```go
func kthSmallest(root *TreeNode, k int) int {
    stack := []*TreeNode{}
    curr := root
    count := 0

    for curr != nil || len(stack) > 0 {
        for curr != nil {
            stack = append(stack, curr)
            curr = curr.Left
        }
        curr = stack[len(stack)-1]
        stack = stack[:len(stack)-1]
        count++
        if count == k {
            return curr.Val
        }
        curr = curr.Right
    }
    return -1
}
```

## 五、从遍历序列重建二叉树

### 5.1 前序 + 中序 → 重建

**原理**：
- 前序的第一个元素一定是根
- 在中序中找到根的位置，左边是左子树，右边是右子树

```go
func buildTree(preorder []int, inorder []int) *TreeNode {
    if len(preorder) == 0 {
        return nil
    }

    rootVal := preorder[0]
    root := &TreeNode{Val: rootVal}

    // 在中序中找根的位置
    mid := 0
    for i, v := range inorder {
        if v == rootVal {
            mid = i
            break
        }
    }

    // 左子树的节点数 = mid
    root.Left = buildTree(preorder[1:1+mid], inorder[:mid])
    root.Right = buildTree(preorder[1+mid:], inorder[mid+1:])
    return root
}
```

## 六、LCA（最近公共祖先）

**题目**：给定二叉树中两个节点 p 和 q，找到它们的最近公共祖先。

```go
func lowestCommonAncestor(root, p, q *TreeNode) *TreeNode {
    if root == nil || root == p || root == q {
        return root
    }
    left := lowestCommonAncestor(root.Left, p, q)
    right := lowestCommonAncestor(root.Right, p, q)

    if left != nil && right != nil {
        return root  // p 和 q 分别在左右子树 → 当前节点就是 LCA
    }
    if left != nil {
        return left  // 都在左边
    }
    return right     // 都在右边
}
```

**思路**：递归地问每个子树"你那边有 p 或 q 吗？"如果一个节点的左右两边各找到一个，那它就是 LCA。

## 速查表

| 问题 | 方法 | 时间复杂度 |
|------|------|----------|
| 遍历（前/中/后/层） | 递归或栈/队列 | O(N) |
| 最大深度 | 递归 | O(N) |
| 是否对称 | 双指针递归 | O(N) |
| 直径 | 后序遍历维护全局最大值 | O(N) |
| BST 查找/插入 | 递归或迭代 | O(log N) 平均 |
| BST 删除 | 递归，找后继替换 | O(log N) 平均 |
| BST 验证 | 递归传递上下界 | O(N) |
| 从遍历重建 | 前序定根，中序分左右 | O(N²) → O(N) 用 map |
| LCA | 分治递归 | O(N) |
