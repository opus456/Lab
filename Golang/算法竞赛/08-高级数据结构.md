## 一、并查集（Union-Find）

### 1.1 核心概念

维护一组不相交集合，支持两个操作：
- **Find(x)**：x 属于哪个集合？
- **Union(x, y)**：合并 x 和 y 所在的集合

### 1.2 Go 实现（路径压缩 + 按秩合并）

```go
type DSU struct {
    parent []int
    rank   []int
}

func NewDSU(n int) *DSU {
    parent := make([]int, n)
    rank := make([]int, n)
    for i := range parent {
        parent[i] = i
    }
    return &DSU{parent, rank}
}

func (d *DSU) Find(x int) int {
    if d.parent[x] != x {
        d.parent[x] = d.Find(d.parent[x]) // 路径压缩
    }
    return d.parent[x]
}

func (d *DSU) Union(x, y int) bool {
    px, py := d.Find(x), d.Find(y)
    if px == py {
        return false // 已经在同一集合
    }
    // 按秩合并
    if d.rank[px] < d.rank[py] {
        px, py = py, px
    }
    d.parent[py] = px
    if d.rank[px] == d.rank[py] {
        d.rank[px]++
    }
    return true
}
```

**时间复杂度**：几乎 O(1)（准确说是 O(α(N))，α 是反阿克曼函数，增长极慢）。

### 1.3 经典应用

```go
// 判断图中是否有环（无向图）
func hasCycle(n int, edges [][2]int) bool {
    dsu := NewDSU(n)
    for _, e := range edges {
        if !dsu.Union(e[0], e[1]) {
            return true // 两端已连通，加这条边会形成环
        }
    }
    return false
}

// 求连通分量个数
func countComponents(n int, edges [][2]int) int {
    dsu := NewDSU(n)
    count := n
    for _, e := range edges {
        if dsu.Union(e[0], e[1]) {
            count--
        }
    }
    return count
}
```

## 二、树状数组（Binary Indexed Tree / Fenwick Tree）

### 2.1 核心概念

支持的操作（1-indexed 数组）：
- **单点修改**：`a[i] += delta`，O(log N)
- **前缀查询**：`sum(a[1..i])`，O(log N)

比线段树代码短很多，常数小。

### 2.2 核心原理

利用二进制的 lowbit（最低位的 1）将数组映射到树状结构上。

```
lowbit(x) = x & (-x)

lowbit(6) = lowbit(110₂) = 010₂ = 2
lowbit(8) = lowbit(1000₂) = 1000₂ = 8
```

### 2.3 Go 实现

```go
type BIT struct {
    tree []int
    n    int
}

func NewBIT(n int) *BIT {
    return &BIT{tree: make([]int, n+1), n: n}
}

// 单点加
func (b *BIT) Update(i, delta int) {
    for ; i <= b.n; i += i & (-i) {
        b.tree[i] += delta
    }
}

// 前缀和 [1, i]
func (b *BIT) Query(i int) int {
    sum := 0
    for ; i > 0; i -= i & (-i) {
        sum += b.tree[i]
    }
    return sum
}

// 区间和 [l, r]
func (b *BIT) RangeQuery(l, r int) int {
    return b.Query(r) - b.Query(l-1)
}
```

### 2.4 经典应用：逆序对

```go
func countInversions(nums []int) int {
    // 离散化
    sorted := make([]int, len(nums))
    copy(sorted, nums)
    sort.Ints(sorted)
    sorted = unique(sorted)
    rank := map[int]int{}
    for i, v := range sorted {
        rank[v] = i + 1
    }

    bit := NewBIT(len(sorted))
    count := 0
    for _, v := range nums {
        r := rank[v]
        count += bit.Query(len(sorted)) - bit.Query(r) // 已出现的比v大的数
        bit.Update(r, 1)
    }
    return count
}
```

## 三、线段树（Segment Tree）

### 3.1 核心概念

对数组维护一棵二叉树，支持：
- **区间查询**（如区间最值、区间和）：O(log N)
- **单点/区间修改**：O(log N)

### 3.2 基础实现（区间和 + 单点修改）

```go
type SegTree struct {
    tree []int
    n    int
}

func NewSegTree(nums []int) *SegTree {
    n := len(nums)
    st := &SegTree{tree: make([]int, 4*n), n: n}
    st.build(nums, 1, 0, n-1)
    return st
}

func (st *SegTree) build(nums []int, node, lo, hi int) {
    if lo == hi {
        st.tree[node] = nums[lo]
        return
    }
    mid := (lo + hi) / 2
    st.build(nums, 2*node, lo, mid)
    st.build(nums, 2*node+1, mid+1, hi)
    st.tree[node] = st.tree[2*node] + st.tree[2*node+1]
}

func (st *SegTree) Update(idx, val int) {
    st.update(1, 0, st.n-1, idx, val)
}

func (st *SegTree) update(node, lo, hi, idx, val int) {
    if lo == hi {
        st.tree[node] = val
        return
    }
    mid := (lo + hi) / 2
    if idx <= mid {
        st.update(2*node, lo, mid, idx, val)
    } else {
        st.update(2*node+1, mid+1, hi, idx, val)
    }
    st.tree[node] = st.tree[2*node] + st.tree[2*node+1]
}

func (st *SegTree) Query(l, r int) int {
    return st.query(1, 0, st.n-1, l, r)
}

func (st *SegTree) query(node, lo, hi, l, r int) int {
    if l <= lo && hi <= r {
        return st.tree[node]
    }
    mid := (lo + hi) / 2
    sum := 0
    if l <= mid {
        sum += st.query(2*node, lo, mid, l, r)
    }
    if r > mid {
        sum += st.query(2*node+1, mid+1, hi, l, r)
    }
    return sum
}
```

### 3.3 懒标记（区间修改）

当需要"区间整体加一个值"时，不要逐个修改，而是在节点上打标记，查询时下推。

```go
type LazySegTree struct {
    tree []int
    lazy []int
    n    int
}

func NewLazySegTree(n int) *LazySegTree {
    return &LazySegTree{
        tree: make([]int, 4*n),
        lazy: make([]int, 4*n),
        n:    n,
    }
}

func (st *LazySegTree) pushDown(node, lo, hi int) {
    if st.lazy[node] != 0 {
        mid := (lo + hi) / 2
        st.tree[2*node] += st.lazy[node] * (mid - lo + 1)
        st.tree[2*node+1] += st.lazy[node] * (hi - mid)
        st.lazy[2*node] += st.lazy[node]
        st.lazy[2*node+1] += st.lazy[node]
        st.lazy[node] = 0
    }
}

func (st *LazySegTree) RangeAdd(l, r, val int) {
    st.rangeAdd(1, 0, st.n-1, l, r, val)
}

func (st *LazySegTree) rangeAdd(node, lo, hi, l, r, val int) {
    if l <= lo && hi <= r {
        st.tree[node] += val * (hi - lo + 1)
        st.lazy[node] += val
        return
    }
    st.pushDown(node, lo, hi)
    mid := (lo + hi) / 2
    if l <= mid { st.rangeAdd(2*node, lo, mid, l, r, val) }
    if r > mid  { st.rangeAdd(2*node+1, mid+1, hi, l, r, val) }
    st.tree[node] = st.tree[2*node] + st.tree[2*node+1]
}
```

## 四、ST 表（Sparse Table）

### 4.1 核心概念

**静态**区间最值查询：预处理 O(N log N)，查询 O(1)。不支持修改。

### 4.2 Go 实现

```go
type SparseTable struct {
    table [][]int
    log2  []int
}

func NewSparseTable(nums []int) *SparseTable {
    n := len(nums)
    k := 0
    for (1 << k) <= n { k++ }

    table := make([][]int, k)
    for i := range table {
        table[i] = make([]int, n)
    }
    copy(table[0], nums) // table[0][i] = nums[i]

    for j := 1; j < k; j++ {
        for i := 0; i+(1<<j)-1 < n; i++ {
            table[j][i] = min(table[j-1][i], table[j-1][i+(1<<(j-1))])
        }
    }

    log2 := make([]int, n+1)
    for i := 2; i <= n; i++ {
        log2[i] = log2[i/2] + 1
    }

    return &SparseTable{table, log2}
}

// O(1) 区间最小值查询
func (st *SparseTable) Query(l, r int) int {
    j := st.log2[r-l+1]
    return min(st.table[j][l], st.table[j][r-(1<<j)+1])
}
```

## 五、数据结构选择指南

| 需求 | 推荐结构 | 修改 | 查询 |
|------|---------|------|------|
| 集合合并/查询 | 并查集 | O(α) | O(α) |
| 前缀和 + 单点修改 | 树状数组 | O(logN) | O(logN) |
| 区间和/最值 + 区间修改 | 线段树(懒标记) | O(logN) | O(logN) |
| 静态区间最值 | ST 表 | 不支持 | O(1) |
