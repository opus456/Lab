# 图论

> **目标**：掌握图的表示方式、遍历算法和经典最短路/最小生成树/拓扑排序算法。

---

## 一、图的基础

### 1.1 表示方法

**邻接矩阵**（适合稠密图）：
```go
graph := make([][]int, n)
for i := range graph {
    graph[i] = make([]int, n)
}
graph[u][v] = weight
```

**邻接表**（适合稀疏图，竞赛最常用）：
```go
type Edge struct{ to, weight int }
adj := make([][]Edge, n)
adj[u] = append(adj[u], Edge{v, w})
adj[v] = append(adj[v], Edge{u, w}) // 无向图
```

### 1.2 BFS（广度优先搜索）

层层扩展，保证第一次到达时距离最短（无权图）。

```go
func bfs(adj [][]int, start int) []int {
    n := len(adj)
    dist := make([]int, n)
    for i := range dist { dist[i] = -1 }
    dist[start] = 0
    queue := []int{start}

    for len(queue) > 0 {
        u := queue[0]
        queue = queue[1:]
        for _, v := range adj[u] {
            if dist[v] == -1 {
                dist[v] = dist[u] + 1
                queue = append(queue, v)
            }
        }
    }
    return dist
}
```

### 1.3 DFS（深度优先搜索）

一条路走到黑，走不通再回头。

```go
func dfs(adj [][]int, u int, visited []bool) {
    visited[u] = true
    for _, v := range adj[u] {
        if !visited[v] {
            dfs(adj, v, visited)
        }
    }
}
```

**DFS 常见应用**：
- 判断连通性
- 计算连通分量个数
- 检测环（有向图用颜色标记，无向图记录父节点）
- 拓扑排序

## 二、最短路算法

### 2.1 Dijkstra（非负权单源最短路）

**核心思想**：每次从未确定的节点中选距离最小的，确定它，然后更新邻居。

```go
func dijkstra(adj [][]Edge, start int) []int {
    n := len(adj)
    dist := make([]int, n)
    for i := range dist { dist[i] = math.MaxInt64 }
    dist[start] = 0

    // 用堆优化
    h := &EdgeHeap{{start, 0}}
    heap.Init(h)

    for h.Len() > 0 {
        cur := heap.Pop(h).(Edge)
        u, d := cur.to, cur.weight
        if d > dist[u] { continue } // 过期的条目

        for _, e := range adj[u] {
            newDist := dist[u] + e.weight
            if newDist < dist[e.to] {
                dist[e.to] = newDist
                heap.Push(h, Edge{e.to, newDist})
            }
        }
    }
    return dist
}
```

**时间复杂度**：O((V+E) log V)

**注意**：Dijkstra 不能处理负权边！有负权用 Bellman-Ford 或 SPFA。

### 2.2 Bellman-Ford（可处理负权）

```go
func bellmanFord(edges [][3]int, n, start int) ([]int, bool) {
    dist := make([]int, n)
    for i := range dist { dist[i] = math.MaxInt64 }
    dist[start] = 0

    for round := 0; round < n-1; round++ {
        for _, e := range edges {
            u, v, w := e[0], e[1], e[2]
            if dist[u] != math.MaxInt64 && dist[u]+w < dist[v] {
                dist[v] = dist[u] + w
            }
        }
    }

    // 检测负权环
    for _, e := range edges {
        u, v, w := e[0], e[1], e[2]
        if dist[u] != math.MaxInt64 && dist[u]+w < dist[v] {
            return dist, true // 有负权环
        }
    }
    return dist, false
}
```

**时间复杂度**：O(VE)

### 2.3 Floyd-Warshall（全源最短路）

```go
func floyd(dist [][]int, n int) {
    for k := 0; k < n; k++ {
        for i := 0; i < n; i++ {
            for j := 0; j < n; j++ {
                if dist[i][k]+dist[k][j] < dist[i][j] {
                    dist[i][j] = dist[i][k] + dist[k][j]
                }
            }
        }
    }
}
```

**时间复杂度**：O(V³)。适合 V ≤ 500。

### 2.4 最短路算法选择

| 场景 | 算法 | 复杂度 |
|------|------|--------|
| 无权图 | BFS | O(V+E) |
| 非负权单源 | Dijkstra + 堆 | O((V+E)logV) |
| 有负权单源 | Bellman-Ford/SPFA | O(VE) |
| 全源最短路 | Floyd | O(V³) |

## 三、最小生成树（MST）

### 3.1 Kruskal（按边排序 + 并查集）

```go
func kruskal(n int, edges [][3]int) int {
    sort.Slice(edges, func(i, j int) bool {
        return edges[i][2] < edges[j][2]
    })

    parent := make([]int, n)
    for i := range parent { parent[i] = i }

    var find func(int) int
    find = func(x int) int {
        if parent[x] != x {
            parent[x] = find(parent[x])
        }
        return parent[x]
    }

    totalWeight := 0
    edgeCount := 0
    for _, e := range edges {
        u, v, w := e[0], e[1], e[2]
        pu, pv := find(u), find(v)
        if pu != pv {
            parent[pu] = pv
            totalWeight += w
            edgeCount++
            if edgeCount == n-1 { break }
        }
    }
    return totalWeight
}
```

### 3.2 Prim（从一个点开始扩展）

```go
func prim(adj [][]Edge, n int) int {
    visited := make([]bool, n)
    h := &EdgeHeap{{0, 0}} // {节点, 权重}
    heap.Init(h)
    totalWeight := 0
    count := 0

    for h.Len() > 0 && count < n {
        cur := heap.Pop(h).(Edge)
        if visited[cur.to] { continue }
        visited[cur.to] = true
        totalWeight += cur.weight
        count++

        for _, e := range adj[cur.to] {
            if !visited[e.to] {
                heap.Push(h, e)
            }
        }
    }
    return totalWeight
}
```

## 四、拓扑排序

### 4.1 BFS（Kahn 算法）

```go
func topologicalSort(n int, edges [][2]int) []int {
    indegree := make([]int, n)
    adj := make([][]int, n)
    for _, e := range edges {
        adj[e[0]] = append(adj[e[0]], e[1])
        indegree[e[1]]++
    }

    queue := []int{}
    for i := 0; i < n; i++ {
        if indegree[i] == 0 {
            queue = append(queue, i)
        }
    }

    order := []int{}
    for len(queue) > 0 {
        u := queue[0]
        queue = queue[1:]
        order = append(order, u)
        for _, v := range adj[u] {
            indegree[v]--
            if indegree[v] == 0 {
                queue = append(queue, v)
            }
        }
    }

    if len(order) != n {
        return nil // 有环
    }
    return order
}
```

### 4.2 判断有向图是否有环

拓扑排序后 `len(order) < n` 就说明有环。

## 五、二分图

### 5.1 判断二分图（染色法）

```go
func isBipartite(adj [][]int) bool {
    n := len(adj)
    color := make([]int, n) // 0=未染色, 1=白, 2=黑

    for i := 0; i < n; i++ {
        if color[i] != 0 { continue }
        queue := []int{i}
        color[i] = 1
        for len(queue) > 0 {
            u := queue[0]
            queue = queue[1:]
            for _, v := range adj[u] {
                if color[v] == 0 {
                    color[v] = 3 - color[u] // 1→2, 2→1
                    queue = append(queue, v)
                } else if color[v] == color[u] {
                    return false
                }
            }
        }
    }
    return true
}
```
