## 一、素数

### 1.1 试除法判素数

```go
func isPrime(n int) bool {
    if n < 2 { return false }
    for i := 2; i*i <= n; i++ {
        if n%i == 0 { return false }
    }
    return true
}
```

### 1.2 埃拉托斯特尼筛法

```go
func sieve(n int) []bool {
    is := make([]bool, n+1)
    for i := 2; i <= n; i++ { is[i] = true }
    for i := 2; i*i <= n; i++ {
        if is[i] {
            for j := i * i; j <= n; j += i {
                is[j] = false
            }
        }
    }
    return is
}
```

**时间复杂度**：O(N log log N)

### 1.3 线性筛（欧拉筛）

每个合数只被最小质因子筛掉一次，严格 O(N)。

```go
func linearSieve(n int) []int {
    primes := []int{}
    isComp := make([]bool, n+1)
    for i := 2; i <= n; i++ {
        if !isComp[i] {
            primes = append(primes, i)
        }
        for _, p := range primes {
            if i*p > n { break }
            isComp[i*p] = true
            if i%p == 0 { break } // 关键：保证每个数只被最小质因子筛
        }
    }
    return primes
}
```

## 二、GCD / LCM / 扩展 GCD

### 2.1 辗转相除法

```go
func gcd(a, b int) int {
    for b != 0 { a, b = b, a%b }
    return a
}

func lcm(a, b int) int {
    return a / gcd(a, b) * b
}
```

### 2.2 扩展欧几里得

求 $ax + by = \gcd(a, b)$ 的一组解。

```go
func extGcd(a, b int) (g, x, y int) {
    if b == 0 {
        return a, 1, 0
    }
    g, x1, y1 := extGcd(b, a%b)
    return g, y1, x1 - (a/b)*y1
}
```

**应用**：求模逆元。当 $\gcd(a, m) = 1$ 时，$a^{-1} \bmod m$ 即 extGcd 返回的 x。

## 三、快速幂 / 模逆元

### 3.1 快速幂

```go
func powmod(base, exp, mod int) int {
    result := 1
    base %= mod
    for exp > 0 {
        if exp&1 == 1 {
            result = result * base % mod
        }
        exp >>= 1
        base = base * base % mod
    }
    return result
}
```

### 3.2 费马小定理求模逆元

当 p 为素数且 $\gcd(a, p) = 1$ 时：

$$a^{-1} \equiv a^{p-2} \pmod{p}$$

```go
func modInverse(a, p int) int {
    return powmod(a, p-2, p)
}
```

## 四、组合数

### 4.1 预处理阶乘

```go
const MOD = 1_000_000_007
var fac, ifac []int

func initComb(n int) {
    fac = make([]int, n+1)
    ifac = make([]int, n+1)
    fac[0] = 1
    for i := 1; i <= n; i++ {
        fac[i] = fac[i-1] * i % MOD
    }
    ifac[n] = powmod(fac[n], MOD-2, MOD)
    for i := n - 1; i >= 0; i-- {
        ifac[i] = ifac[i+1] * (i + 1) % MOD
    }
}

func C(n, k int) int {
    if k < 0 || k > n { return 0 }
    return fac[n] % MOD * ifac[k] % MOD * ifac[n-k] % MOD
}
```

### 4.2 Pascal 三角形（小范围）

```go
func buildPascal(n int) [][]int {
    c := make([][]int, n+1)
    for i := 0; i <= n; i++ {
        c[i] = make([]int, n+1)
        c[i][0] = 1
        for j := 1; j <= i; j++ {
            c[i][j] = (c[i-1][j-1] + c[i-1][j]) % MOD
        }
    }
    return c
}
```

## 五、常用数论技巧

### 5.1 模运算规则

```
(a + b) % m = ((a % m) + (b % m)) % m
(a * b) % m = ((a % m) * (b % m)) % m
(a - b) % m = ((a % m) - (b % m) + m) % m   // 注意加 m 防负
(a / b) % m = (a * b⁻¹) % m                  // 除法用逆元
```

### 5.2 质因数分解

```go
func factorize(n int) map[int]int {
    factors := map[int]int{}
    for p := 2; p*p <= n; p++ {
        for n%p == 0 {
            factors[p]++
            n /= p
        }
    }
    if n > 1 {
        factors[n]++
    }
    return factors
}
```

### 5.3 矩阵快速幂

用于加速线性递推。斐波那契数列 O(log N) 求第 N 项。

```go
type Matrix [2][2]int

func matMul(a, b Matrix, mod int) Matrix {
    var c Matrix
    for i := 0; i < 2; i++ {
        for j := 0; j < 2; j++ {
            for k := 0; k < 2; k++ {
                c[i][j] = (c[i][j] + a[i][k]*b[k][j]) % mod
            }
        }
    }
    return c
}

func matPow(m Matrix, n, mod int) Matrix {
    result := Matrix{{1, 0}, {0, 1}} // 单位矩阵
    for n > 0 {
        if n&1 == 1 {
            result = matMul(result, m, mod)
        }
        m = matMul(m, m, mod)
        n >>= 1
    }
    return result
}

// fib(n) % mod  O(log n)
func fibFast(n, mod int) int {
    if n <= 1 { return n }
    m := Matrix{{1, 1}, {1, 0}}
    r := matPow(m, n-1, mod)
    return r[0][0]
}
```

## 速查表

| 工具 | 代码核心 | 复杂度 |
|------|---------|--------|
| 判素数 | `for i*i<=n` | O(√N) |
| 筛素数 | 埃筛/欧拉筛 | O(N log log N) / O(N) |
| GCD | `for b!=0 {a,b=b,a%b}` | O(log N) |
| 快速幂 | `base*base%mod` | O(log N) |
| 模逆元 | `powmod(a, p-2, p)` | O(log P) |
| 组合数 | 预处理阶乘+逆元 | O(1) 查询 |
| 矩阵快速幂 | 矩阵乘法+快速幂 | O(K³ log N) |
