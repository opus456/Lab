## 学习路线建议

**零基础入门**：项目工程 00 → 01 → 02 → 03 → 04

**想打算法竞赛**：算法竞赛 00 → 01 → 03 → 04 → 06（先掌握核心）→ 其余按需

**想做后端开发**：项目工程全部按顺序 → 然后算法竞赛选读

**想快速复习**：每篇文档开头都有"速查表"部分
