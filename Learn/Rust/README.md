# Rust 学习文档

## 为什么学 Rust

Rust 是一门系统级编程语言，兼顾性能、安全和并发。它没有垃圾回收器，却能在编译时保证内存安全。从 2016 年起连续多年被 Stack Overflow 评为"最受喜爱的编程语言"。

Rust 适用于：高性能服务器、命令行工具、WebAssembly、嵌入式系统、操作系统、游戏引擎、区块链等领域。

## 学习路线建议

**初学者（有其他语言基础）：**
00 → 01 → 02（重点！多花时间）→ 03 → 04 → 05 → 06（重点）→ 07（难点，可先跳过细节）→ 08 → 09 → 15 → 17（做个小工具建立信心）→ 10 → 11 → 12 → 16 → 18

**有 C/C++ 经验：**
00 → 01 → 02（对比 C++ RAII）→ 03 → 04 → 06 → 07 → 05 → 10 → 11 → 12 → 14 → 13 → 16/17 → 18

## 前置知识

- 至少掌握一门编程语言（Python/JavaScript/C/Java 均可）
- 了解基本的数据结构（数组、链表、哈希表）
- 了解基本的命令行操作

## 标记说明

- ⭐ 重点：核心知识，必须深入理解
- ⭐⭐ 核心难点：Rust 独有的难点，需要反复练习
- 高级（不常用）：了解即可，实际开发中较少直接使用
- 实战：偏应用，结合框架和工具
