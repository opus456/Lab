## 1. 结构体 (Struct)

### 基本结构体

```rust
struct User {
    username: String,
    email: String,
    age: u32,
    active: bool,
}

fn main() {
    let user = User {
        username: String::from("alice"),
        email: String::from("alice@example.com"),
        age: 30,
        active: true,
    };
    println!("{} ({})", user.username, user.email);

    // 可变实例
    let mut user2 = User {
        username: String::from("bob"),
        email: String::from("bob@example.com"),
        age: 25,
        active: true,
    };
    user2.age = 26; // 整个实例要么可变要么不可变
}
```

### 字段简写和结构体更新语法

```rust
fn create_user(username: String, email: String) -> User {
    User {
        username,   // 字段名和变量名相同时可以简写
        email,
        age: 0,
        active: true,
    }
}

fn main() {
    let user1 = create_user(String::from("alice"), String::from("a@b.com"));
    // 结构体更新语法
    let user2 = User {
        email: String::from("new@email.com"),
        ..user1  // 其余字段从 user1 移动/复制
    };
    // 注意：user1.username 被移动了，user1 部分失效
}
```

### 方法和关联函数 (impl)

```rust
struct Rectangle {
    width: f64,
    height: f64,
}

impl Rectangle {
    // 关联函数（构造器）
    fn new(width: f64, height: f64) -> Self {
        Rectangle { width, height }
    }

    fn square(size: f64) -> Self {
        Rectangle { width: size, height: size }
    }

    // 方法：&self 不可变借用
    fn area(&self) -> f64 {
        self.width * self.height
    }

    // &mut self 可变借用
    fn scale(&mut self, factor: f64) {
        self.width *= factor;
        self.height *= factor;
    }

    // self 获取所有权（消耗自身）
    fn into_square(self) -> Rectangle {
        let side = (self.width + self.height) / 2.0;
        Rectangle { width: side, height: side }
    }
}

fn main() {
    let mut rect = Rectangle::new(10.0, 5.0);
    println!("Area: {}", rect.area());
    rect.scale(2.0);
    let square = rect.into_square(); // rect 被消耗
}
```

**self 的三种形式：**
- `&self`：只读访问，最常用
- `&mut self`：需要修改自身
- `self`：获取所有权，用于转换操作

## 2. 枚举 (Enum)

### 携带数据的枚举（Rust 的杀手特性）

```rust
enum Message {
    Quit,
    Move { x: i32, y: i32 },
    Write(String),
    ChangeColor(u8, u8, u8),
}

impl Message {
    fn process(&self) {
        match self {
            Message::Quit => println!("Quitting"),
            Message::Move { x, y } => println!("Moving to ({}, {})", x, y),
            Message::Write(text) => println!("Writing: {}", text),
            Message::ChangeColor(r, g, b) => {
                println!("Color: #{:02x}{:02x}{:02x}", r, g, b)
            }
        }
    }
}
```

**为什么 Rust 的枚举如此强大？**

它是"代数数据类型"（ADT），能表达"值是 A 或 B 或 C"，每种情况携带不同数据。用于状态机、错误类型、消息系统、AST 等。

### Option<T> —— 空值的安全替代

```rust
// Rust 没有 null！用 Option 表示"可能没有值"
fn find_user(id: u32) -> Option<String> {
    if id == 1 { Some(String::from("Alice")) } else { None }
}

fn main() {
    // 必须处理 None，编译器强制你考虑
    match find_user(1) {
        Some(name) => println!("Found: {}", name),
        None => println!("Not found"),
    }

    // 常用简写
    let name = find_user(2).unwrap_or(String::from("Unknown"));

    // if let：只关心一种情况
    if let Some(name) = find_user(1) {
        println!("Hello, {}!", name);
    }
}
```

## 3. 模式匹配

### match 表达式

```rust
fn describe_number(n: i32) -> &'static str {
    match n {
        0 => "zero",
        1..=9 => "single digit",
        10..=99 => "double digit",
        _ => "large number",
    }
}
```

### 解构和守卫

```rust
enum Shape {
    Circle { radius: f64 },
    Rectangle { width: f64, height: f64 },
}

fn area(shape: &Shape) -> f64 {
    match shape {
        Shape::Circle { radius } => std::f64::consts::PI * radius * radius,
        Shape::Rectangle { width, height } => width * height,
    }
}

// 守卫条件
fn classify(temp: f64) {
    match temp {
        t if t < 0.0 => println!("Freezing"),
        t if t < 30.0 => println!("Normal"),
        _ => println!("Hot"),
    }
}
```

### if let / while let / let-else

```rust
fn main() {
    // if let
    let value = Some(42);
    if let Some(v) = value {
        println!("Got: {}", v);
    }

    // while let
    let mut stack = vec![1, 2, 3];
    while let Some(top) = stack.pop() {
        println!("Popped: {}", top);
    }

    // let-else (Rust 1.65+)
    let input = "42";
    let Ok(num) = input.parse::<u32>() else {
        println!("Parse failed");
        return;
    };
    println!("Parsed: {}", num);
}
```

## 4. 实际工程案例：状态机

```rust
enum ConnectionState {
    Disconnected,
    Connecting { attempt: u32 },
    Connected { session_id: String },
    Error { message: String },
}

enum Event {
    Connect,
    Success(String),
    Failure(String),
    Disconnect,
}

impl ConnectionState {
    fn next(self, event: Event) -> Self {
        match (self, event) {
            (ConnectionState::Disconnected, Event::Connect) =>
                ConnectionState::Connecting { attempt: 1 },
            (ConnectionState::Connecting { .. }, Event::Success(id)) =>
                ConnectionState::Connected { session_id: id },
            (ConnectionState::Connecting { attempt }, Event::Failure(msg)) if attempt < 3 =>
                ConnectionState::Connecting { attempt: attempt + 1 },
            (ConnectionState::Connecting { .. }, Event::Failure(msg)) =>
                ConnectionState::Error { message: msg },
            (ConnectionState::Connected { .. }, Event::Disconnect) =>
                ConnectionState::Disconnected,
            (state, _) => state,
        }
    }
}
```

- `struct Contact { name, phone, email: Option<String> }`
- `struct AddressBook { contacts: Vec<Contact> }`
- 实现增删查改，搜索返回 `Vec<&Contact>`
- 注意所有权和借用的正确使用

### 练习 3（工程实践）：配置文件解析器

解析 INI 格式配置：
- `enum ConfigValue { Str(String), Int(i64), Float(f64), Bool(bool) }`
- 支持 `[section]` 和 `key = value`
- 返回 `Result` 处理解析错误

### 练习 4（挑战）：表达式求值器

- 定义 AST：`enum Expr { Num(f64), Add(Box<Expr>, Box<Expr>), ... }`
- 实现 `eval() -> Result<f64, String>`
- 实现 `display() -> String`
- 可选：解析字符串为 AST

**思路提示：**
- 练习 1：每个变体携带必要数据，用 match 分别计算

- 练习 4：`Box<Expr>` 因为递归类型需要间接引用。eval 用递归 match
