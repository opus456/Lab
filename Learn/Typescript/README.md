# TypeScript 学习文档

## 简介

TypeScript 是由微软开发的 JavaScript 超集，通过引入静态类型系统，在编译阶段捕获潜在错误，提供更强大的 IDE 支持和代码重构能力。对于中大型项目和团队协作而言，TypeScript 能显著提升代码的可维护性和开发效率。

## 学习建议

**初学者路线（有 JavaScript 基础）：**

1. 按顺序学习 00 ~ 04（基础篇），打好类型系统基础
2. 完成 05 ~ 06，理解面向对象与高级类型
3. 学习 08 ~ 09，掌握模块与工程配置
4. 根据使用的框架选读 12 ~ 14

**有经验的开发者：**

1. 快速浏览 00 ~ 04 查漏补缺
2. 重点攻克 06 ~ 07（高级类型与类型体操）
3. 深入 09 ~ 10（工程化与声明文件）
4. 学习 11（设计模式），提升架构能力
5. 按需阅读框架篇
6. 完成 15 的综合实战，并从 16 选择 2–3 个项目做源码精读

## 前置知识

- JavaScript ES6+ 基础语法
- 了解 Node.js 和 npm 基本使用
- 具备任意一款代码编辑器的使用经验（推荐 VS Code）
